# Olist E-commerce Business Analysis Dashboard

This project analyzes Olist’s Brazilian e-commerce dataset in Power BI to evaluate sales performance, customer behavior, seller performance, and delivery issues.

## Project Overview
The dashboard was built to answer key business questions around:
- Revenue performance
- Customer retention
- Seller contribution
- Delivery delays and customer satisfaction

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Dataset
- Source: Olist Brazilian E-commerce Public Dataset/ kaggle
- Period: 2016–2018
- Scope: ~100K orders

## Key Insights
- 98.67K orders, 96.10K customers, 3,095 sellers
- $13.59M total revenue
- Returning customer rate: 3%
- On-time orders received much higher ratings than late orders
- Most delays were related to shipping/logistics rather than sellers
